GitHub Pages templates often look generic because they rely on a default theme and very little page structure. A few deliberate design choices can change that quickly.

This layout leans on serif headlines, tighter spacing, and a warm editorial palette to create a more distinctive feel. The result is still lightweight, but it carries more identity than a standard starter theme.

## Keep the workflow easy

The goal is not to build a complicated publishing system. The goal is to make a repo feel presentable, readable, and easy to update.

With Jekyll, new stories automatically appear in the homepage feed and archive as soon as you add another post file.
